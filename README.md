# 📸 Social Login Page

A modern, responsive social media-inspired login page built with **HTML5, CSS3, and Vanilla JavaScript**. The project features a clean UI, smooth animations, client-side validation, and a premium user experience. It is designed as an original implementation inspired by modern social media design patterns, without copying any specific brand's proprietary assets.

---

## ✨ Features

- 🎨 Modern and responsive design
- 📱 Mobile-first layout
- 🔒 Username/Email and Password login form
- 👁️ Show/Hide password toggle
- ✅ Client-side form validation
- ⚡ Loading animation during login
- 🎉 Success animation after validation
- 🌈 Smooth hover and fade-in effects
- 💎 Glassmorphism login card
- 🌊 Floating animated background elements
- 🔥 Google Fonts (Poppins)
- 🎯 Clean, semantic, and well-commented code

## 🚀 Getting Started

1. Clone the repository.

```bash
git clone https://github.com/yourusername/social-login-page.git
```

2. Navigate to the project folder.

```bash
cd social-login-page
```

3. Open `index.html` in your preferred browser.

No installation or build tools are required.
